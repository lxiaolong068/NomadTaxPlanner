### 5.2 核心功能详细设计

#### 5.2.1 税务居民身份判断器

**功能目标**：帮助用户在5分钟内初步了解自己可能在哪些国家被视为税务居民

**用户流程**：
```
开始
  ↓
Q1: 你的国籍是？ [选择国家]
  ↓
Q2: 过去12个月你在哪些国家停留过？ [多选+天数]
  ↓
Q3: 你有固定居所的国家？ [选择]
  ↓
Q4: 你的主要收入来源类型？ [雇员/自由职业/公司]
  ↓
Q5: 你的雇主/客户主要在哪里？ [选择]
  ↓
[算法处理]
  ↓
结果页：
- 可能的税务居民身份国家列表
- 每个国家的风险等级（高/中/低）
- 建议的下一步行动
- 相关国家指南链接
- "这只是初步判断"的免责声明
```

**边界情况处理**：
- 复杂情况（如多国籍）→ 建议咨询专业人士
- 信息不足 → 提示补充信息
- 高风险场景 → 强调寻求专业帮助

#### 5.2.2 FEIE资格检测器

**功能目标**：帮助美国公民判断是否符合海外收入免税额度

**核心逻辑**：
1. 物理存在测试 (Physical Presence Test)
   - 在任意连续12个月内，至少330天在美国境外
   - 工具自动计算符合条件的时间段

2. 真实居住测试 (Bona Fide Residence Test)
   - 在外国建立真实居住意图
   - 整个纳税年度居住在外国

**输出内容**：
- 是否符合FEIE资格
- 可排除的最高金额（2024年为$126,500）
- 需要填写的税表（Form 2555）
- 常见错误警告

#### 5.2.3 停留天数追踪器

**功能目标**：帮助用户精确记录和追踪在各国的停留天数

**核心功能**：
- 手动输入行程（到达日期、离开日期、国家）
- 日历视图可视化
- 自动计算各国累计天数
- 183天规则预警（接近阈值时提醒）
- 数据导出（CSV/PDF）

**数据存储**：
- 本地存储（LocalStorage）用于未登录用户
- 云同步（登录用户）

## 13. 风险控制

### 13.1 法律风险

| 风险 | 缓解措施 |
|------|---------|
| 税务建议责任 | 清晰的免责声明，强调"仅供参考，不构成专业税务建议" |
| 信息准确性 | 注明信息来源和更新日期，建议用户核实官方信息 |
| 数据隐私 | GDPR合规，隐私政策，用户数据加密 |
| 联盟披露 | FTC合规，所有联盟链接明确标注 |

### 13.2 免责声明模板

> **重要提示**：NomadTaxPlanner提供的所有信息和工具仅供教育和参考目的，不构成专业的税务、法律或财务建议。税务法规复杂且因个人情况而异，信息可能随时变化。在做出任何税务决策之前，请务必咨询合格的税务专业人士。使用本网站即表示您同意我们的服务条款和免责声明。

### 13.3 运营风险

| 风险 | 缓解措施 |
|------|---------|
| 信息过时 | 建立定期审查机制，监控政策变化 |
| 技术故障 | 定期备份，监控告警 |
| 竞争加剧 | 持续创新，建立社区护城河 |
| 依赖单一流量来源 | 多元化流量渠道 |

### 13.4 财务风险

- 控制初期投入（使用免费/低成本工具）
- 变现前确保现金流可支撑运营
- 设定止损点（如6个月无起色则复盘调整）
